## Overview
Broadstripes' layouts let you customize the contents of the search results panel. With **flexible contact info columns**, you can can decide on the fly which contact info you want displayed and what type of contact data you donâ€™t want to see. You can also create multiple flexible contact info columns to hold different information. For instance, you can set up one column to display home and personal phone numbers, and another to show work email addresses.
